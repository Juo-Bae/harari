# 변경 이력

## [1.0.0] - 2024년

### 완료된 기능

#### Phase 1: 기본 설정
- ✅ Next.js 프로젝트 초기화 (TypeScript, Tailwind CSS)
- ✅ Google Sheets API 연동 설정
- ✅ 기본 레이아웃 및 라우팅 구성

#### Phase 2: 인증 시스템
- ✅ AUTH 시트 연동
- ✅ 로그인/로그아웃 기능
- ✅ 토큰 기반 자동 로그인
- ✅ 비밀번호 변경 팝업

#### Phase 3: 데이터 관리
- ✅ InventoryContext 생성
- ✅ 재고 데이터 로딩 API
- ✅ 재고조사 데이터 로딩 API

#### Phase 4: 페이지 구현
- ✅ 홈 화면
- ✅ 마이페이지 (사용자 정보, 비밀번호 변경, 로그아웃)
- ✅ 재고현황 페이지 (테이블 형태)
- ✅ 재고조사 페이지 (수정/업데이트 기능)

#### Phase 5: 업데이트 기능
- ✅ 재고조사 데이터 업데이트
- ✅ 재고 시트 자동 업데이트
- ✅ 재고로그 시트 자동 업데이트 (중복 처리, 정렬)

#### Phase 6: 테스트 및 최적화
- ✅ 에러 처리 개선
- ✅ UI/UX 개선
- ✅ 성능 최적화 (useCallback, useMemo)

### 주요 기능

1. **인증 시스템**
   - 이름과 번호로 로그인
   - 자동 로그인 (토큰 기반)
   - 비밀번호 변경 (첫 로그인 시 필수)

2. **재고 관리**
   - 재고 현황 조회
   - 재고조사 및 업데이트
   - 재고 변경 이력 자동 기록

3. **사용자 관리**
   - 마이페이지에서 사용자 정보 확인
   - 비밀번호 변경
   - 로그아웃

### 기술 스택

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Google Sheets API
- React Context API
